# next-dress-eshop

> Next.js Dress eShop

in-progress Ecommerce responsive made with:
  - `Next.js`
  - `TypeScript`
  - `Redux`
  - `Redux-persist`
  - `Hooks`
  - `SCSS`
  - `BEM`

```bash
$ yarn install

$ yarn start
```

Visit http://localhost:3000

```yml
tested nodejs version: 16.16.0
```

## pages

- Home page: `/`
- Products page: `/products`
- Product single page: `/product/:id`
- Cart page: `/cart`
- Login page: `/login`
- Register page: `/register`
- 404 page: `/page-not-found`
